# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-19

### Added
- Initial release of `gmnd` Discord Bot.
- Core interaction logic using `gmn` CLI.
- Context management with file locking and daily summarization.
- Support for multimodal input (file attachments).
- Slash commands: `/status`, `/config`, `/set_system`, `/clear_context`, `/model`, `/help`.
- Transitioned dependency management from `pip` to `uv`.
- Added `system.log` and graceful shutdown support.
- Comprehensive documentation: `README.md`, `docs/instructions.md`.
